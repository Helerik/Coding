Combining Selectors

We are able to combine selectors. For example, doing p.big {...} would apply the style to all paragraphs with the big class.

Another way of combining selectors is applying the styling to a direct child of a certain tag, by doing something like "p > span". For example this span piece of text, which is inside a paragraph.
